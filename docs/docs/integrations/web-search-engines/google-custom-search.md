# Google Custom Search

https://programmablesearchengine.google.com/

## Maven Dependency

```xml

<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-web-search-engine-google-custom</artifactId>
    <version>1.0.1-beta6</version>
</dependency>
```

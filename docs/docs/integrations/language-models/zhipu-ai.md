# Zhipu AI

- https://www.zhipuai.cn/
- https://open.bigmodel.cn/


## Maven Dependency

```xml
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-zhipu-ai</artifactId>
    <version>0.32.0</version>
</dependency>
```

- `ZhipuAiChatModel`
- `ZhipuAiStreamingChatModel`

# LocalAI

https://localai.io/


## Maven Dependency

```xml
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-local-ai</artifactId>
    <version>0.35.0</version>
</dependency>
```


## APIs

- `LocalAiChatModel`
- `LocalAiStreamingChatModel`
- `LocalAiLanguageModel`
- `LocalAiStreamingLanguageModel`

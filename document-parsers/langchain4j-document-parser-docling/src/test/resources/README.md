# Test Resources

This directory contains sample documents used for integration testing.

## Sample Documents Needed

For comprehensive integration testing, add the following sample files:

- `sample.pdf` - Simple PDF with text content
- `sample.docx` - Simple Word document with text
- `sample-with-tables.pdf` - PDF containing tables for table extraction testing
- `sample-scanned.pdf` - Scanned PDF for OCR testing

## Creating Sample Documents

Sample documents should be:

- Small in size (< 100KB each)
- Contain simple, non-sensitive test content
- Cover different document features (text, tables, images)

## Future Enhancement

These files will be used by integration tests once docling-serve testcontainers setup is complete.

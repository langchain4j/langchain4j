# LangChain4j Document Parser - Docling

Integration of [Docling](https://github.com/docling-project/docling-java) document parser into LangChain4j.

## Status

**Work in Progress** - This module is currently under development.

## Overview

Docling is an advanced document processing library that provides:
- Advanced PDF understanding with layout analysis
- Table structure recognition
- OCR support for scanned documents
- Formula extraction
- Support for PDF, DOCX, PPTX, XLSX, and more

## Planned Features

Basic document parsing
Metadata extraction
Configuration options (OCR, table extraction modes)
Integration tests
Examples
